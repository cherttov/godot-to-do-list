# Godot To-Do-List #
Godot plugin that adds a to-do-list tab in the engine.
## Configuration ##
After installing the plugin, turn it on in the settings and restart the Godot Engine. Once any task is added 
a `to-do-list.tres` file is going to be created in the root folder. 
The `to-do-list.tres` file saves all the tasks and theirs status.
> **Note:** Do not delete the `to-do-list.tres` file, unless you want to get rid of all of the existing tasks.
